# 机械工程本科生 编程自学路线

## 1. 基础阶段：入门编程思维

### 1.1 选择入门语言
- **推荐：Python**（语法简单，应用广泛）
- MATLAB（课程或实验常用，可同步学习）

### 1.2 Python 基础
- 变量、数据类型（int, float, str, list, dict, tuple）
- 条件语句（if/elif/else）
- 循环（for, while）
- 函数定义与调用
- 文件读写（txt、csv）

#### 推荐资源
- [菜鸟教程 Python3](https://www.runoob.com/python3/python3-tutorial.html)
- [Python 官方教程](https://docs.python.org/zh-cn/3/tutorial/index.html)
- B站：Python 入门课程

## 2. 进阶阶段：工程常用技能

### 2.1 数值计算与数据分析
- 安装并使用 **NumPy**（数组、矩阵运算）
- 使用 **Pandas** 处理表格数据（CSV/Excel）
- 使用 **Matplotlib** 画图（实验数据可视化）

#### 推荐资源
- [NumPy 中文文档](https://numpy.org.cn/)
- [Pandas 官方文档](https://pandas.pydata.org/docs/)
- B站：NumPy、Pandas、Matplotlib 教程

### 2.2 MATLAB（如课程要求）
- 基本语法、矩阵运算、m文件
- 绘图、仿真、Simulink 简介

## 3. 应用阶段：工程计算与自动化

### 3.1 工程算法实现
- 编写数值微分、积分、线性方程求解、最小二乘拟合等常用算法
- 用代码处理实验数据、绘制曲线、做误差分析

### 3.2 工程仿真
- 用 Python/MATLAB 实现动力学、热学、流体等简单模型仿真
- 了解有限元分析（可用 ANSYS/ABAQUS 软件，掌握数据导入导出）

### 3.3 自动化与批量处理
- 用代码批量处理实验数据、自动生成报告
- 批量读取、处理、可视化大量数据文件

## 4. 进阶提升（选修/项目驱动）

### 4.1 计算机视觉与图像处理
- 学习 **OpenCV**，实现基本的图像处理、特征提取、物体检测

### 4.2 机器人与自动化
- 学习 Arduino/Raspberry Pi 编程基础
- 了解 ROS 基础知识

### 4.3 机器学习初步（选学）
- 用 scikit-learn 做简单的回归、分类实验

## 5. 工程开发与协作实践

### 5.1 版本管理工具
- 学习 **Git/GitHub** 的基本用法（clone、commit、push、pull、branch）
- 参加开源项目或小组项目，体验团队协作

### 5.2 代码规范与自学能力
- 学会写有注释、结构清晰的代码
- 善于查阅官方文档、Stack Overflow、GitHub issues

---

## 推荐学习顺序

1. Python 基础
2. NumPy、Pandas、Matplotlib
3. MATLAB 基础（如课程要求）
4. 工程仿真与算法实现
5. 自动化与批量处理
6. 进阶领域：OpenCV、Arduino、Git
7. 实践项目（课程/兴趣驱动）

## 附加建议

- 多做小项目、小工具，把学到的技能用在真实需求上
- 主动和同学、老师交流，参与实验室或工程竞赛
- 保持好奇心，敢于自学和解决问题
